# CytoscapeActionbar

## Description
Random layout for Cytoscape.js nodes

## Dependencies
 * Cytoscape.js >= 2.2
 
## Installation
Require package using bower:

```
bower install iisg/cytoscape-random-preset-layout --save
```

## Usage
Call the layout with
```
cy.layout(
    name: 'presetWithRandom'
)
```
